Change Blindness with Java

A faculty member had been using this website http://gocognitive.net/demo/change-blindness, developed by Dr. Steffen Werner of the University of Idaho, to demonstrate change blindness. Unfortunately the website used Adobe Flash which was discontinued in most of the world in 2020.

Prior to the site going down, its function was analyzed and some of the images it used captured. This Java program is an attempt to replicate its function.
